# AutoBrain 🧠🚗
AI Vehicle Manager para Subaru Forester 2023 y Mazda CX-5 2017

## Acceder a la app
👉 [Abrir AutoBrain](https://eriklirab-coder.github.io/autobrain/autobrain.html)

## Funciones
- 💬 Chat con mecánico IA
- 📊 Dashboard con alertas
- ✅ Mantenimiento con recordatorios
- ⚠️ Análisis de códigos OBD
- ⛽ Escaneo de tickets de gasolina
- 📷 Escaneo de tablero/odómetro
- 📋 Documentos del vehículo con escaneo IA
- 🔩 Registro fotográfico de piezas
- 💾 Datos guardados permanentemente
